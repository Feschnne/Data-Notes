# Notes - Máster en Data Science & Big Data

Carpetas que recogen apuntes, ejercicios y proyectos que voy desarrollando durante el máster. Éstas organizadas por áreas técnicas.
El objetivo de este repositorio es mostrar mi progreso y aprendizaje en Data Science, y sirve como base para futuros proyectos y portfolios.

## Cómo usarlo
Cada carpeta contiene notebooks con apuntes y con ejemplos prácticos, ejercicios y plantillas.
Todos los notebooks intentan seguir la misma plantilla de nota técnica para mantener consistencia.
